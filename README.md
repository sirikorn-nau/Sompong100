# What's the hurry, bro?

A mobile application for reporting motorcycle taxi services

## Tech Stack :
- **Frontend:** React Native, Javascript  
- **Database:** Firebase
- **Version Control:** Git and GitHub
- **Code Editor and tools**: VS Code

## Getting Started
Install dependencies
```
npm install
```
